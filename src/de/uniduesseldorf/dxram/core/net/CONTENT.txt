#Classes
AbstractConnection - Model of a network connection
AbstractConnectionCreator - Model of a network connection creator (initializes a new network connection)
AbstractMessage - Model of a network message
AbstractRequest - Model of a network request
AbstractResponse - Model of a network response
BufferCache - Cache for multiple ByteBuffer
ConnectionManager - Handles network connections
MessageDirectory - Class for registering network messages
NetworkHandler - Default implementation of the NetworkInterface
NetworkInterface - API of the network service
NIOConnectionCreator - Connection creator for TCP connections using Java NIO
RequestMap - Stores open requests
RequestStatistic - Request-Statistic
TaskExecutor - Executor class for the network tasks
ThroughputStatistic - Statistic of the network throughput
