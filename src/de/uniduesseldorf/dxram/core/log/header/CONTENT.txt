#Classes
AbstractLogEntryHeader - Model of a log entry header
DefaultPrimLogEntryHeader - Extends AbstractLogEntryHeader for a normal log entry header (primary log)
DefaultPrimLogTombstone - Extends AbstractLogEntryHeader for a default tombstone (primary log)
DefaultSecLogEntryHeader - Extends AbstractLogEntryHeader for a normal log entry header (secondary log)
DefaultSecLogTomstone - Extends AbstractLogEntryHeader for a normal tombstone (secondary log)
MigrationPrimLogEntryHeader - Extends AbstractLogEntryHeader for a migration log entry header (primary log)
MigrationPrimLogTombstone - Extends AbstractLogEntryHeader for a migration tombstone (primary log)
MigrationSecLogEntryHeader - Extends AbstractLogEntryHeader for a migration log entry header (secondary log)
MigrationSecLogTomstone - Extends AbstractLogEntryHeader for a migration tombstone (secondary log)
